Rust Programming Language

Reference Book : The Rust Programming Language

Installation

Install latest version of rust on linux

$ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh   
$ rustc --version

We will summarize the programming concepts specific to Rust in this page.

  • metadata and toolchains : rustup
  • rustc : compiler
  • cargo (build & package manager) : cargo

Cargo​

Cargo is Rust's build system and Package manager. Helps in building your code, downloading the libraries your code depends on, and building those libraries/dependencies.

  • cargo new : create project
  • cargo build : creates an executable file in target/debug directory
  • cargo run : compiles the code and then runs the resultant executable
  • cargo check : checks if program still compiles
  • Cargo stores the build in the target/debug directory
  • cargo build --release : compile with optimizations, an executable in target/release

Guessing Game Example​

use std::cmp::Ordering;
use std::io;
use rand::Rng;

fn main() {
    println!("Guess the number!");

    let secret_number = rand::rng().random_range(1..=100);
    
    //println!("The secret number is: {secret_number}");

    loop{
        
        println!("Please input your guess.");

        let mut guess = String::new();

        io::stdin()
	    .read_line(&mut guess)
            .expect("Failed to read line");

        println!("You guessed: {guess}");

        let guess: u32 = match guess.trim().parse() {
            Ok(num) => num,
            Err(_) => continue,
        };

        match guess.cmp(&secret_number){ 
            Ordering::Less => println!("Too small!"),
            Ordering::Greater => println!("Too big!"),
            Ordering::Equal => {
                println!("You win!");
                break;
            }
        }
    }
}
  • If a type you want to use isn’t in the prelude, you have to bring that type into scope explicitly with a use statement.
  • println! is a macro that prints a string to the screen
  • Storing values with variables : let mut guess = String::new();
    • let is used to create a variable
    • mut makes the variable mutable
    • :: is an operator that allows us to namespace the new function call to String
    • new is an associated function of the String type
  • variables are immutable by default, meaning once we give the variable a value, the value won’t change.
  • String is a growable, heap-allocated data structure owned by its variable.
  • & indicates that this argument is a reference, which gives you a way to let multiple parts of your code access one piece of data without needing to copy that data into memory multiple times.
